# ISPU Jakarta - K-Means Clustering

Proyek ini menganalisis kualitas udara Jakarta menggunakan metode K-Means Clustering 
pada dataset ISPU (Indeks Standar Pencemar Udara) dengan 14.577 data dari 5 stasiun pemantauan.

## Tools
- Python, Google Colab
- Pandas, Scikit-learn, Matplotlib, Seaborn

## Hasil
Data dikelompokkan menjadi 3 cluster:
- Cluster Polusi Rendah
- Cluster Polusi Sedang  
- Cluster Polusi Tinggi
